# EBACProject01
Meu primeiro projeto Git
Escrito em phyton faz operações matemáticas.
Este é um projeto que calcula o seno, cosseno e a tangente de um ângulo dado pelo usuário.

Para executar o arquivo no linux use:
Dê a permissão necessária para executar o arquivo: chmod +x desafio18.sh
Depois execute: ./desafio18.sh

Para executar no Windows use:
Método 1: Usando o Git Bash
Baixe e instale o Git para Windows, que inclui o terminal Git Bash. 
Abra a pasta onde está o seu arquivo .sh.
Clique com o botão direito do mouse em um espaço vazio e escolha Git Bash Here.
No terminal que se abrir, digite o seguinte comando e aperte Enter:
bashbash desafio18.sh
Método 2: Usando o WSL.
Se o seu script depende de comandos específicos do sistema Linux (como gerenciadores de pacotes ou interações profundas com o sistema):Habilite o WSL no seu painel de controle (Recursos do Windows) ou execute wsl --install no PowerShell.
Abra o terminal do WSL/Ubuntu.
Navegue até a pasta do seu arquivo (por exemplo, cd /mnt/c/PastaDoSeuArquivo).
Execute o script digitando:bashbash desafio18.sh
